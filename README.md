## Getting Started

How to use :-<br />
Swipe left -> to delete the todo<br />
Swipe right -> to mark todo as completed<br />
Tap on todo card -> to edit the current todo<br /> 

Data Storage :-<br />
This app uses SQLite to store data locally on your android device.
